Hugo+GitHub搭建博客

发表于 2021-04-06 更新于 2021-05-10 分类于 Hugo
  1. 下载安装（Windows10 64位）
  • 在https://github.com/gohugoio/hugo/releases下载对应缩文件，放在特定目录。
  • 以管理员身份打开cmd，进入解压后的目录(任意位置输入D:进入D盘），输入hugo version 测试是否成功

  1. 生成myblog
    输入命令hugo new site myblog该文件为博客主目录

  2. 下载主题
    在themes目录下，运行git clone https://github.com/sfengyuan/edidor下载主题

  3. 本地启动博客
    进入myblog文件夹

    访问http://localhost:1313本地测试

  4. 部署到GitHub

    • 创建以 昵称.github.io 的GitHub项目，如 joscodhua.github.io 。
    • 在myblog目录下，运行hugo --theme=edidor --baseUrl="https://joscodehua.github.io" --buildDrafts产生public文件，public文件下的内容会上传到GitHub
    • 进入public文件，运行下列命令

  5. 其他备注

    • 配置文件：config.toml

    • Logo文件必须是 svg 格式, 以后会修改, 复制到 根目录/static/images/ 目录里. 你可以提供3种版本:logo.svg logo_light.svg logo_dark.svg

    • 文件内说明

      title: “JavaSE中的容器”
      date: 2019-07-08T18:25:28+08:00
      draft: true
      categories: [“test1”] //文件分类
      tags: [“test1”, “test2”] //标签

    • Hugo导入图片

      • 图片资源放置在根目录下的static/images下，也就是网站的绝对路径
      • 图片存放在content/post下与博客文件名相同文件夹下，图片请求路径要和博客文件名相同

    • markdown图片居中
